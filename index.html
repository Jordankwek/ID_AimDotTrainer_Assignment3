Aim Dot Trainer

Aim Dot Trainer is a game designed to improve players aim in First-person shooting games such as Valorant, Counter-Strike : GO, OverWatch and PUBG. With the different modes available, it will allow them to improve their different aspects of aiming.

About the Game

The various modes are tailored to improve different aspects of FPS gaming. Modes such as sprint and survival improves the players' tracking of enemies in game while other modes such as blitz help in deathmatches where the purpose is to kill as many enemies as possible
